# Project Blueprint

## Overview

A simple web application with a login feature.

## Current State

*   **index.html**: Basic HTML structure with a title and a placeholder button.
*   **style.css**: Empty stylesheet.
*   **main.js**: Empty JavaScript file.

## Plan for Login Feature

1.  **HTML**: Create a login form in `index.html` with username and password fields, and a "Login" button.
2.  **CSS**: Add styles to `style.css` to make the login form visually appealing and centered on the page.
3.  **JavaScript**: In `main.js`, add an event listener to the form to handle submission. Initially, it will log the form data to the console.
